# E-commerce Back End Starter Code
This app is an amaizing e-commerce tool that will be help any small store to track their inventory, pices and more

## Install
Create database in MySQL 
  mysql -u root -p
  source db/schema.sql

Seed you database
  In the command line 
    node seeds/index.js

## Usage
node server.js
